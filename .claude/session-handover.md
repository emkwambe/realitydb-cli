## Dogfood Pipeline (Proven)

```
# One-command refresh
.\tools\refresh-sandbox.ps1

# Manual pipeline
realitydb pack export --template saas --records 1000 --output ./packs --name saas
node tools/pack-to-sql.js ./packs/saas.realitydb-pack.json public/data/saas.sql

# Direct generator (bypasses pack serialization issues)
cd tools
node generate-template-sql.js 1000 ../path/to/output.sql saas
```
